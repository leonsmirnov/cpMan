## Capture tips

- Use a **clean desktop** (hide clutter) or a soft wallpaper — templates add their own background, but the raw capture should be **window-only** or small region.
- **Dark mode** or **Light mode**: pick one and use it for all captures (Light mode often reads better in marketing).
- Capture at **2x Retina** if possible; templates scale down to 2880 x 1800.
- **Menu bar**: include the cpMan icon in shots 01–04 if it fits naturally.
- Do **not** show personal clipboard data — demo mode uses fictional samples only.

## Minimum upload size

Export final slides at **2880 x 1800**. App Store Connect accepts this and scales for smaller displays. You can also export **1280 x 800** if file size matters — keep 16:10 aspect ratio.
