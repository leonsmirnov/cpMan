# Age-rating questionnaire answers

Apple's questionnaire lives in **App Store Connect → App Information →
Age Rating**. With the answers below, cpMan is rated **4+** in every
territory.

Every question gets **None** unless explicitly noted otherwise.

### Frequency categories

| Question | Answer |
|----------|--------|
| Cartoon or Fantasy Violence | None |
| Realistic Violence | None |
| Prolonged Graphic or Sadistic Realistic Violence | None |
| Profanity or Crude Humor | None |
| Mature/Suggestive Themes | None |
| Horror/Fear Themes | None |
| Medical/Treatment Information | None |
| Alcohol, Tobacco, or Drug Use or References | None |
| Sexual Content or Nudity | None |
| Graphic Sexual Content and Nudity | None |
| Simulated Gambling | None |
| Contests | None |

### Boolean toggles

| Question | Answer |
|----------|--------|
| Unrestricted Web Access | **No** (cpMan has no networking at all). |
| Gambling | No |
| User-Generated Content | No (the only "content" is the user's own clipboard text, stored locally and never displayed to anyone else). |
| Messaging or Social Networking features | No |
| In-app purchase | No |

### Final rating

After saving the above answers, App Store Connect will display:

> **Rating: 4+**

That is the desired result. No further action is needed in this section.
