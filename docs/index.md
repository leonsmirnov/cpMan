# cpMan

cpMan is a minimal menu-bar clipboard manager for macOS. It remembers
the last 100 things you copied and brings any of them back with a
single keystroke.

- Press **⌃⌥V** anywhere on macOS to open the picker.
- Filter by typing, press **Return** to copy the selection back to the
  clipboard, then paste with **⌘V** as usual.
- Plain text only. No accounts. No network. No system permissions.
